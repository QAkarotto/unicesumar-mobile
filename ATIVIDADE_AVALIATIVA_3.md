# Trabalho 3 - Aplicação de Calculadora em Flutter utilizando Componentização

**Valor: 2,5 pontos**

## Objetivo

Aplicar conceitos de **Flutter e desenvolvimento de interfaces** utilizando **componentização de widgets**.

## Descrição da Atividade

O grupo deverá desenvolver um aplicativo de **calculadora simples** utilizando o framework Flutter.

O aplicativo deve possuir uma interface gráfica funcional e permitir a realização de operações matemáticas básicas.

## Funcionalidades Mínimas

A calculadora deve permitir:

- inserção de números
- operações matemáticas:
  - adição
  - subtração
  - multiplicação
  - divisão
- exibição do resultado na tela
- limpeza da operação.

## Requisitos Técnicos

O projeto deve utilizar:

- Flutter
- separação da interface em **widgets reutilizáveis**.

Exemplos de componentes possíveis:

- widget para botão numérico
- widget para botão de operação
- widget para display da calculadora.

## Layout

O layout pode utilizar widgets como:

- `Row`
- `Column`
- `Expanded`
- `Container`.

## Documentação (README)

O README deve conter:

- descrição da aplicação
- explicação da estrutura de componentes
- instruções para execução
- imagens ou prints da interface.
