# aula_widget

Projeto Flutter de apoio para aulas focadas em widgets, layouts e componentização.

## Executar o Projeto

```bash
flutter pub get
flutter run
```

## Executar Testes

```bash
flutter test
```

## Uso em Aula

Este projeto é utilizado para:

- praticar composição de widgets;
- experimentar variações de layout;
- separar componentes reutilizaveis;
- comparar abordagens de construção de UI.

## Sugestão de Prática

1. Reproduza o exemplo apresentado em aula.
2. Crie uma variação com novos componentes.
3. Refatore para melhorar legibilidade e reuso.
